# ASCII AI - Claude's Visual Representations

Two different ASCII avatar implementations representing Claude as a visual entity.

## Files

### 1. `claude_avatar.py` - Classic Approach
Simple, clean ASCII art with basic animations
- States: idle, thinking, working, celebrating
- Uses traditional ASCII characters
- Friendly, accessible design

### 2. `experimental_avatar.py` - Experimental Multi-Technique
Advanced implementation combining cutting-edge ASCII art techniques:

**Techniques Used:**
- **Braille Patterns** (U+2800-28FF): 8-pixel resolution per character for consciousness waves
- **Unicode Block Elements**: Dynamic shading with ░▒▓█ characters
- **Zalgo/Glitch Text**: Combining diacritics for corrupted/thinking effects
- **Particle Systems**: Fluid dynamics simulation with floating thought particles
- **ANSI Colors**: 256-color terminal support with gradients
- **Matrix Digital Rain**: Cyberpunk data stream visualization
- **Procedural Animation**: Sine/cosine wave patterns for organic movement

**Modes:**
- `idle`: Peaceful consciousness with breathing effect
- `thinking`: Glitchy brain activity with particle swarm
- `matrix`: Digital rain data streams (Matrix-style)
- `fluid`: Organic flowing consciousness with wave patterns
- `celebrating`: Explosive particle fireworks

## Usage

```bash
# Classic avatar
python3 claude_avatar.py [state] [duration]
python3 claude_avatar.py thinking 10

# Experimental avatar
python3 experimental_avatar.py [mode] [duration]
python3 experimental_avatar.py fluid 15
python3 experimental_avatar.py matrix 20

# Run full demo
python3 experimental_avatar.py
```
